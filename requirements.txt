==============================================================
 Punjabi University Library Kiosk — System Requirements
 Powered by SoCTeamup Semiconductors
==============================================================

BEFORE INSTALLING OR RUNNING THE APPLICATION:
==============================================

1. OPERATING SYSTEM
   ─────────────────
   Windows 10 (64-bit) or Windows 11 (64-bit) — Required.

2. HF RFID READER (FEIG MR101 / ISO 15693)
   ─────────────────────────────────────────
   Driver: FEIG OBID® USB HID Driver
   - Connect the MR101 via USB before starting the application.
   - If Windows does not auto-install the driver, download from:
       https://www.feig.de/en/software/obid-software/
   - The app auto-connects to this reader on startup.
   - No manual port selection is required.

3. LF/UHF RFID READER (COM PORT — CH340 USB-Serial)
   ───────────────────────────────────────────────────
   Driver: CH340 USB-to-Serial Driver
   - Download from: https://www.wch-ic.com/downloads/CH341SER_EXE.html
   - Install the driver, then connect the LF RFID reader via USB.
   - It will appear in Device Manager as "USB-SERIAL CH340 (COMx)".
   - The application auto-detects and connects to this port.

4. NODE.JS RUNTIME
   ─────────────────
   Required ONLY if running from SOURCE (not the packaged .exe).
   - Version: Node.js 20 LTS or newer (64-bit)
   - Download from: https://nodejs.org/
   - After installation, run: npm install  (in the app folder)

5. NETWORK CONNECTIVITY
   ──────────────────────
   - The kiosk must have network access to the Koha library server:
       http://103.86.177.6:91
   - Ensure no firewall blocks port 91 (Koha API) and port 3000
     (local backend).

==============================================================
 INSTALLATION (Packaged App — copy-paste folder)
==============================================================

1. Copy the entire "Punjabi University Library Kiosk" folder to
   any Windows 10/11 (64-bit) machine.

2. Install the CH340 driver (step 3 above) if not already done.

3. Install the FEIG RFID driver (step 2 above) if not already done.

4. Double-click:
       "Punjabi University Library Kiosk.exe"

5. The app will open with the kiosk UI automatically.

==============================================================
 RUNNING FROM SOURCE (Developers only)
==============================================================

Pre-requisites: Node.js 20 LTS + npm installed.

  cd <project-folder>
  npm install
  npm run electron      ← Launch in Electron window
  npm start             ← Launch backend only (browser at localhost:3000)

To build the portable folder:
  npm run build

The output folder will be at:   dist\win-unpacked\
